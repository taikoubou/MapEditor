MapEditor_var_1.0
java必須(バージョンは1.6.5以上？)

MapEditor使い方
・MapEditor_var_1.0.exeを起動します。
・マウスの操作でほぼ操作できます。
・マップを新規作成する場合には【new map】を押してください。
・ファイルを読み込む場合には【import map】を押してください。
・ファイルを保存する場合には【save】を押してください。
・画面が描画されなかった場合には【repaint】を押してください。
・キャラクターの座標を入力する場合はすべての項目に整数を入力した状態で【set】ボタンを押してください。

諸注意
・ファイルを保存する場合には拡張子(xxx【.map】)まで入力してください。
・キャラクターの座標を入力する際、座標を変更させないキャラクターがいるときは、X座標、Y座標ともに[-1]を入力した状態にしてください。
・公式で配布されているサーバーのフォルダ全体に直接保存できないので注意してください。(なぜｗｗｗｗ)
・学校のPCで使う場合は「MapEditor_var_1.0_school.exe」の方を使ってください。
↑普通に起動します。

Q＆A
Q:すこし動作が遅い気がする
A:仕様です

Q:ドラッグすると時々描画されない
A:スピードを落としてドラッグしてください

バグが発生したら、実行手順とエラーメッセージを記録して情報処理部の共有フォルダかどっかにまとめておいてください。定期的に確認しに行きます。(たぶん)
